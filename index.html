Frozen Lake

Github Link

Training a bot to play Frozen Lake - v0 game, which has 4 locations (S - starting point, H - hole, F - frozen surface, G - goal). The agent controls the movement of a character in a grid world. Some tiles of the grid are walkable, and others lead to the agent falling into the water. Additionally, the movement direction of the agent is uncertain and only partially depends on the chosen direction. The agent is rewarded for finding a walkable path to a goal tile.
